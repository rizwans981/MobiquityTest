Weather APP

As a part of Mobiquity code challange a weather app has been developed. Using this application one could search for a city and its weather conditions.These are the following steps to use the App

Home Screen:

The home screen has two features one is a searchbar with a map.and a list that contains user search results, by default the search list will be empty.

Using search bar search for desired city and select the search location, you will observe a pin will be located on the map in the home screen and the city name in the search results view

Click on the city name in the list on home screen, it will direct to weather details screen of the particular city that the user has selected.

Weather Details Screen:

This screen contains the weather details of the city like humidity, cloud conditions, temparatures.

Instructions Screen:

This screen contains the instructions of how to use this app.
